# Visual Asset & Graphics Standards

All graphics created in the Px Content Studio or for @PxJourney must follow these rules:

## Core Aesthetic
- Retro pixel + CRT + neon cyan/pink/magenta palette
- Press Start 2P or similar pixel font
- Strong contrast, scanlines, and subtle glows
- "PIXAL EYES ONLY" and survivor mint references are encouraged

## Pixal Graphics (from template 781515)
- Always use the live AtomicAssets fetch in the Studio
- Preferred frame styles: Neon (default), CRT, Pixel Border, Clean
- Never stretch or distort the Pixal image
- Always include date + mint number overlay when possible

## General Rules
- Keep file sizes reasonable for X
- Prefer PNG for graphics with transparency
- Consistent use of the design-system color tokens

All visuals should feel like they belong to the same Pixel Journey universe.
