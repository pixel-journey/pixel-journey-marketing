# standards/

This folder extends the existing `pixel-journey-standards` and design-system marketing guidelines with marketing-specific rules for the SoMe engine.

## Files

- `brand-voice-and-messaging-standards.md` — Tone, personality, emoji usage, pixel-perfect language
- `social-media-community-content-standards.md` — Posting cadence, thread structure, CTA rules
- `visual-asset-and-graphics-standards.md` — How to use Canvas frames + Pixal PFPs
- `ai-and-generated-content-guidelines.md` — How to safely use LLMs while staying on-brand

All standards here take precedence for @PxJourney output.
