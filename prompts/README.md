# prompts/

This folder contains reusable system prompts for LLMs (Antigravity, Claude, Grok, etc.) to help scale Pixel Journey content while staying perfectly on-brand.

## How to Use

Copy the content of a `.txt` file and paste it as the system prompt in your AI tool, then give it a task like:
- "Turn these patch notes into a 5-tweet thread"
- "Generate 8 hook variations for this topic"
- "Create a variation of this post in the Pixel Journey voice"

All prompts are designed to produce output that matches our tone, references verifiable on-chain mechanics when relevant, and aligns with the 6 pillars.
